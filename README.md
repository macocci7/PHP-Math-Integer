# PHP-Math-Integer

Math library of Integer:

PHP-Math-Integer treats the subjects of number theory (only natural number).

## Requirements

- PHP 8.0.0 (CLI) or later
- Composer

## Installation

```bash
composer require macocci7/php-math-integer
```

## Usage

## Functions

- Number
- Prime
- Divisor
- Euclid (Algorithm)
- Fraction
